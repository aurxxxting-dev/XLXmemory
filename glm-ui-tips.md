# GLM 生成漂亮 UI 的 Prompt 技巧

## 核心原则
GLM 是逻辑型模型，需要**明确的视觉指令**才能生成好看的 UI。不要期待它像 Gemini 一样有设计直觉。

---

## 技巧 1：指定设计系统

**❌ 模糊请求**
```
帮我写一个登录页面
```

**✅ 明确指定**
```
用 Tailwind CSS 写一个登录页面，风格参考 Material Design 3：
- 圆角按钮（rounded-lg）
- 阴影效果（shadow-md）
- 主色调使用 Indigo-600
- 输入框有浮动标签效果
```

---

## 技巧 2：给具体的设计参考

**✅ 有效 Prompt**
```
设计一个仪表盘，参考以下风格：
- 深色模式（bg-slate-900）
- 卡片式布局（rounded-xl, bg-slate-800）
- 渐变色强调元素（from-blue-500 to-purple-600）
- 字体使用 Inter
- 间距遵循 8px 网格系统
```

---

## 技巧 3：组件化思维

**✅ 先定义设计规范**
```
先定义一套设计系统：

【颜色】
- 主色：#6366f1（Indigo）
- 背景：#0f172a（Slate 900）
- 卡片：#1e293b（Slate 800）
- 文字：#f8fafc（Slate 50）

【间距】
- 小：8px
- 中：16px
- 大：24px

【圆角】
- 按钮：8px
- 卡片：16px

然后基于此设计一个导航栏组件...
```

## 技巧 4：提供参考链接

**✅ 让 GLM 模仿现有设计**
```
参考 https://dribbble.com/shots/xxxx 的设计风格，
用 React + Tailwind 实现一个类似的个人主页，
要求：
1. 相同的配色方案
2. 类似的卡片阴影效果
3. 响应式布局
```

---

## 技巧 5：分步迭代

**不要一次要完美，分阶段优化：**

**第一步：功能框架**
```
先实现一个基础的文章列表页面，只要功能，不追求美观
```

**第二步：添加样式**
```
现在优化这个页面：
1. 添加卡片阴影和悬停效果
2. 使用更现代的字体
3. 优化间距和对齐
4. 添加渐变背景
```

**第三步：细节打磨**
```
继续优化：
1. 添加加载动画
2. 优化移动端显示
3. 添加微交互（按钮点击效果）
```

---

## 技巧 6：使用成熟组件库

**✅ 直接指定组件库**
```
使用 Ant Design 组件库实现一个后台管理界面：
- 左侧导航菜单
- 顶部面包屑
- 中间数据表格
- 要求支持暗色主题
```

**推荐组件库（告诉 GLM 直接用）：**
- Tailwind UI
- Ant Design
- Element Plus
- Material UI
- Chakra UI

## 技巧 8：明确视觉层次

**✅ 结构化描述**
```
设计一个 pricing 页面，视觉层次如下：

【第一层级 - 标题区】
- 大标题：48px，bold，居中
- 副标题：20px，gray-600，居中

【第二层级 - 卡片区】
- 3 列卡片布局
- 每个卡片：
  - 顶部：套餐名称 + 价格（价格用大号字体）
  - 中部：功能列表（带勾选图标）
  - 底部：CTA 按钮（Primary 颜色）

【第三层级 - 页脚】
- 信任标识（logo 墙）
- FAQ 链接
```

## 实战模板

### 通用 UI Prompt 模板

```
【项目类型】：xxx
【技术栈】：React + Tailwind CSS
【设计风格】：参考 xxx（Material Design / Apple / 具体网站）
【配色方案】：
- 主色：xxx
- 背景：xxx
- 强调色：xxx

【页面结构】：
1. 导航栏（固定顶部，透明背景滚动变实色）
2. 英雄区（大标题 + CTA 按钮 + 背景图）
3. 功能展示（3 列卡片，带图标）
4. 页脚

【交互要求】：
- 悬停效果
- 滚动动画
- 响应式适配

【其他要求】：
- 使用现代字体（Inter / system-ui）
- 8px 网格系统
- 圆角统一 8px 或 16px
```

---

## 总结

| Gemini | GLM |
|--------|-----|
| 给模糊需求也能做好 | 需要精确指令 |
| 有设计直觉 | 需要明确规范 |
| 一次成型 | 建议分步迭代 |

**用 GLM 做 UI 的黄金法则：**
> 不要问"好不好看"，要告诉它"具体怎么做"
